# AI_Job_Marker_Analysis
📈 Global Data Science & AI Salary Analysis (2026)
Exploring the Financial Impact of Machine Learning & Remote Work
This project is a deep dive into the 2026 job market for data professionals. Using a dataset of over 10,000 global job postings, I analyzed the correlation between specific technical skills (Python, SQL, ML, Deep Learning) and total compensation.

🎯 Project Objectives
Identify High-Value Skills: Determine which technologies, such as Deep Learning and Cloud, command the highest salary premiums.
Compare Career Paths: Analyze the salary gap between baseline roles (Data Analyst) and specialized engineering roles (AI Engineer).
Remote Work Trends: Investigate if choosing Remote or Hybrid work models leads to a "salary penalty" across different experience levels.
📊 Data Insights:-
The AI Premium: My analysis shows that Deep Learning (0.24) and Machine Learning (0.23) have the strongest positive correlation with salary increases.

Salary Distribution: While Entry Level roles typically range from $40k to $150k, Senior roles consistently push toward the $200k mark, regardless of whether the role is Remote or Onsite.

Top Earners: AI Engineers and Machine Learning Engineers are the highest-paid roles in the 2026 market, often exceeding $130k+ on average.


🛠️ Technical Stack:-
Analysis: Python (Pandas, NumPy).
Visualization: Seaborn (Heatmaps) and Plotly (Interactive Boxplots/Bar Charts).
Environment: Developed using Google Colab on a mobile-first workflow.

📂 Key Dataset Features:-
job_title: The specific role (e.g., Data Scientist, AI Engineer).
skills_ml / skills_deep_learning: Binary indicators of specialized technical knowledge.
remote_type: Work model (Remote, Onsite, Hybrid).
salary: The total annual compensation (Target Variable).
